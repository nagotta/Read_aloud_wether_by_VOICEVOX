# VOICEVOXキャラによる天気予報お知らせシステム

ずんだもん(VOICEVOXキャラ達)が天気予報をお知らせしてくれるシステムを作成しました。

ずんだもん固定のバージョンとVOICEVOX搭載キャラがランダムに選出されるバージョンがあります。
main.pyから切り替えることが可能です。
